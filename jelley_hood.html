Challenge 1

How Circular Counting Works

If guests sit in a straight LINE, we count every possible order.

But around a CIRCLE, rotating everyone together does NOT create a new arrangement.

Arrangement A

A → B → C → D

Rotated Version

D → A → B → C

These are actually the SAME circular arrangement because everyone kept the same neighbors.

Now consider mirror images:

Clockwise

A → B → C → D

Mirror Image

A → D → C → B

These are reflections of each other. If mirror images count as the same arrangement, we divide by 2.
